<h1 align="center">Spotify-Control</h1>
<p align="center">A snippet to control Spotify.</p>

> [!NOTE]
> Control Spotify in background  
> Click the tray icon to instant toggle Play / Pause  
> Spotify can be hidden in tray!  
> OnScreenDisplay (OSD)  
> No active processes running, very lightweight

- Mute / Unmute
- Volume + / -
- Play / Pause
- Current Playing
- Toogle Full screen
